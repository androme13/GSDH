Hardware compatibility

Compatibility is tracked by brand and model so users can immediately see the current validation level for their device.

How to read this page

  • Full support: validated in normal use and recommended for end users
  • Partial support: usable, but some features or behaviors still need validation
  • Unknown: not yet tested or not enough feedback available

Device families

  • Mirabox
  • Ajazz
  • Soomfon
  • Other compatible clone or rebranded StreamDock-style devices

Important notes

  • Actual behavior may vary depending on firmware version, hardware revision, and Windows environment
  • Direct USB HID access can behave differently across seemingly similar models
  • Simultaneous active control with the vendor application is not officially guaranteed
